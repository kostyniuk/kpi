# CA_Lab5

### *Valid arguments:*

   • *-h, --help to show help*
   
   • *-l, --list to set the array of digits*

   • *-f, --file to set the file*

   • *-v, --version to set the version of an application*
   

### To use the project:
*git clone `https://github.com/kostyniuk/kpi.git`*

*cmake ~/ac-5-lab*

*make*
